# Interview Coding Tasks

A collection of coding and reasoning tasks used in technical interviews.

Each task is self-contained under `tasks/<task-name>/` with its own README and data.

## Tasks

- [`user-behavior-anomaly-detection`](./tasks/user-behavior-anomaly-detection/) — 用户行为异常检测（数据分析 + 业务推理）

## For Candidates

Pick the task the interviewer points you to, read the README, and work on it with any tools you prefer (Python, Excel, Claude, Cursor, ChatGPT, etc.). The evaluation focuses on your **reasoning path**, not code volume or tool choice.
